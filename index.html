Face Recognition based Online Exam

NodeJS Docker Kubernetes Microservice GCP MongoDB YAML

Face Recognition based Online Exam

In COVID situation, many universities might be wondering how they should take online exam which is authenticated and more reliable. To resolve this conundrum, thought of developing a microservice based web application which authenticate student based upon registered face ID and then he can start his exam. Used NodeJS (ES6) as a back-end and MongoDB to fetch random question per student from data store. Dockerized each service layer and deployed them on GCP Kubernetes where each layer horizontally scalable, capabale to perform load balancing.
